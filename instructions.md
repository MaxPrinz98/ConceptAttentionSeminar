We will investigate which object categories are most challenging for the model to generate and extend the analysis to attribute concepts—including color, texture, and spatial relations—using Concept Attention. Beyond evaluating correctly generated objects, we will also study failure cases and hallucinations to identify which concepts are especially difficult for the model to represent.
If time permits, we will compare these findings against Attention Chains (interpreted as discrete-time Markov processes) to explore complementary explanations.
For evaluation, object concepts will be assessed using IoU against ground-truth masks. For attribute concepts, we will begin with manual inspection on a curated subset of samples and then scale the evaluation using vision–language models (VLMs).
